Bootcamp 2021: Planets

My first programming project, concluding the 2 week Programming Bootcamp at City (2021). This Processing project demonstrates an understanding of the use of loops, variables and images etc. I created a sketch about planets. In the sketch, when the position of the mouse hovers over each planet, information about the planet displays on the screen. I aligned the planets and adjusted the sizes whilst also using if loops to determine what pieces of information should be the output for a particular mouse position.

Check out the code on Github: https://github.com/Ahmed-Db/Bootcamp_Project
